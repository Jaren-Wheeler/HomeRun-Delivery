# HomeRun Delivery (Initial Version)
## overview
A peer-to-peer delivery service allowing purchasers to submit delivery jobs for any item purchased on platforms such as Facebook Marketplace, Kijiji, or Ebay. Everyday people, who are verified as drivers,
can accept jobs posted near there location and perform the deliveries.
*** This is the initial development of the project. The updated version has been re-designed and is curently in development in the Jaren-Wheeler/HomeRun repository.
